Created this small demo Project with UV.

**uv init**

create a virtual environment.

**uv venv**

activate the virtual env

Added a requirements.txt folder with dependencies required for langchain framework and wokring with groq models


**uv add -r requirements.txt**

created a account in groq api website.
created a new file inside the AI folder with name agentinvoke.py

run the python file. ---> python name of the file.py
